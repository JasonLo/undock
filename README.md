#Undock

This is a terminal user interface (TUI) for managing Docker containers. It provides an interactive way to view and control your Docker environment directly from the terminal.

## Features

- Automatically detects current repo's compose file and displays the services defined in it.
- Displays all containers (running and stopped), visually differentiating compose-defined services from unrelated containers.

## Stack

- textual
- docker
- pyyaml
